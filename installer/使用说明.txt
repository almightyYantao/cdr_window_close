【插件功能】
  自动处理打开CDR文件时的错误提示,特别是"无效的轮廓ID"错误
  自动点击"忽略"按钮,无需手动操作

【系统要求】
  ✓ Windows 7/8/10/11 (32位或64位)
  ✓ CorelDRAW X7 或更高版本
  ✓ .NET Framework 4.7.2 或更高版本

【安装步骤】非常简单!

  1. 右键点击 "install.bat"
  2. 选择 "以管理员身份运行"
  3. 等待安装完成(约5秒)
  4. 启动或重启 CorelDRAW

  ✓ 就这么简单!

【验证安装】

  启动CorelDRAW后,应该看到提示消息:
  "CorelDRAW自动忽略错误插件已加载"

  如果没有看到:
  1. 打开 CorelDRAW
  2. 工具 → 选项 → 工作区 → 自动化
  3. 点击 "加载/卸载加载项"
  4. 确认插件已在列表中

【使用方法】

  安装后无需任何操作!

  当你打开CDR文件时遇到错误提示,插件会自动:
  ✓ 检测错误对话框
  ✓ 自动点击"忽略"按钮
  ✓ 继续打开文件

  你只需要像往常一样打开文件即可!

【卸载方法】

  1. 右键点击 "uninstall.bat"
  2. 选择 "以管理员身份运行"
  3. 确认卸载
  4. 重启 CorelDRAW

【常见问题】

  Q: 安装后没有任何提示?
  A: 检查是否以管理员权限运行了install.bat

  Q: 提示缺少 .NET Framework?
  A: 下载安装: https://dotnet.microsoft.com/download/dotnet-framework/net472

  Q: CorelDRAW没有自动加载插件?
  A: 手动配置:
     工具 → 选项 → 工作区 → 自动化 → 加载/卸载加载项
     添加插件DLL文件

  Q: 错误对话框没有自动关闭?
  A: 可能是对话框文本不匹配,请联系开发者提供错误截图

  Q: 插件会影响CorelDRAW性能吗?
  A: 不会,插件使用独立线程后台运行,不影响主程序

  Q: 插件安全吗?
  A: 完全安全,开源代码,仅用于窗口自动化,不收集任何数据
